Problem of the Day - Reverse a Doubly Linked List

Link - https://www.geeksforgeeks.org/problems/reverse-a-doubly-linked-list/1

Question - Given a doubly linked list of n elements. Your task is to reverse the doubly linked list in-place.

Example 1:

Input:
LinkedList: 3 <--> 4 <--> 5
Output: 5 4 3
Example 2:

Input:
LinkedList: 75 <--> 122 <--> 59 <--> 196
Output: 196 59 122 75

Expected Time Complexity: O(n).
Expected Auxiliary Space: O(1).

Constraints:
1 <= number of nodes <= 104
0 <= value of nodes <= 104
