Problem of the Day - Height of Binary Tree.

Link of the Problem - https://practice.geeksforgeeks.org/problems/height-of-binary-tree/1

Question - Given a binary tree, find its height.

Example 1:

Input:
     1
    /  \
   2    3
Output: 2
Example 2:

Input:
  2
   \
    1
   /
 3
Output: 3   

Expected Time Complexity: O(N)
Expected Auxiliary Space: O(N)

Constraints:
1 <= Number of nodes <= 105
1 <= Data of a node <= 109
